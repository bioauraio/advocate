# cases/ — материалы дел

Каждое дело — отдельная папка `cases/<номер_дела>/`. Кладите сюда:

- `00_БАЗА_ЗНАНИЙ.md` — консолидированная база дела: факты, линии защиты,
  процессуальная последовательность, реестр документов, выученные уроки.
  Агент начинает работу по делу с чтения этого файла.
- Транскрипции томов дела (markdown). Большие файлы агент читает не целиком,
  а навигацией Grep + Read по диапазонам строк.
- `research/` — практика, статистика, справки по составу.

Содержимое дел — чувствительные данные, в git они не попадают (см. .gitignore).
