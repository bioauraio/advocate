# Судебная статистика РФ: назначенные наказания по статьям УК (2016–2024)

Файл: `sudstat_nakazaniya_po_statyam_2016-2024.csv`
Источник: stat.апи-пресс.рф (Агентство правовой информации, данные Суддепартамента
при ВС РФ), раздел «Данные о назначенном наказании по статьям УК», вся Россия.
Выгружено: 2026-07-19. 8499 строк — каждая статья/часть УК за каждый год 2016–2024.

## Колонки
Год; Статья_норм (нормализованная, напр. `171.1 ч.4`, итог года — `ВСЕГО`);
Показатель (оригинальная подпись сайта); Осуждено; Оправдано; Лишение свободы;
Условное лишение свободы; Ограничение свободы; Штраф; Исправительные работы;
Обязательные работы; Принудительные меры к невменяемым.

Числа = количество осуждённых лиц по основному (наиболее тяжкому) составу.
«Осуждено» может превышать сумму видов наказаний (часть мер — арест, лишение
права занимать должности как основное и др. — сайт в эту таблицу не включает).

## Как пользоваться
- Ищи по колонке Статья_норм: `Grep "171.1 ч.4" <файл>` — оригинальные подписи
  сайта разнятся по годам («часть 4 УК РФ» / «ч. 4» / «часть 4»), нормализованная
  колонка единая.
- Это главный источник для ответа «что реально дают по статье X»: доля реального
  лишения свободы, условки, штрафа; динамика по годам; шанс оправдания.
- Всегда указывай год(ы) и что это федеральные данные по РФ в целом.
