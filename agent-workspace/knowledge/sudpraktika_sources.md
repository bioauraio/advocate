# Источники судебной практики РФ (проверено 2026-07-19)

Справочник для агента и владельца: где брать тексты решений и свежую практику,
что доступно с нашего сервера напрямую (✅) и что капризничает (⚠️).

## Тексты судебных актов

### ✅ sudact.ru — Судебные и нормативные акты РФ (ОСНОВНОЙ)
Крупнейшая бесплатная база текстов: приговоры, решения, определения всех
инстанций (ВС, кассация, апелляция, районные суды, мировые судьи, арбитраж).
- Поиск по тексту, статье закона, суду, региону, дате, виду документа.
- Агрегированные подборки «практика по статье»: `sudact.ru/practice/po-st-171.1-uk-rf/`
  (подставь номер своей статьи).
- Поиск приговоров по статье: `sudact.ru/regular/doc/?regular-txt=ст.+171.1+УК+РФ`
  (+ фильтры вид документа/инстанция/дата).
- С нашего сервера отдаёт результаты без капчи → МОЖНО автоматизировать
  регулярную выгрузку свежих дел по списку статей (cron-скрипт).

### ✅ mos-gorsud.ru — Мосгорсуд
Поиск дел и тексты актов Мосгорсуда и районных судов Москвы. Хорошо для
московской практики; есть раздел открытых данных.

### ✅ kad.arbitr.ru — Картотека арбитражных дел
Все арбитражные дела: карточки, документы, движение дела. Есть неофициальное
JSON-API (используется самим сайтом) — подходит для автоматизации по ИНН/номеру дела.

### ⚠️ ГАС «Правосудие» (bsr.sudrf.ru) — официальный банк решений СОЮ
Официальный первоисточник по судам общей юрисдикции, полнотекстовый поиск.
Работает нестабильно (с нашего сервера — таймауты), тексты часто с задержкой
и обезличены сильнее sudact. Пользоваться через браузер, когда нужен именно
официальный источник.

### ⚠️ vsrf.ru — Верховный Суд РФ
Обзоры судебной практики Президиума ВС (выходят регулярно, это обязательное
чтение), постановления Пленума, обзоры по отдельным категориям дел.
Прямой curl отдаёт 403/404 (анти-бот) — забирать через браузер, либо те же
обзоры брать с sudact/КонсультантПлюс через несколько дней после выхода.

## Статистика (что реально назначают суды)

### ✅ stat.апи-пресс.рф — АПИ «Судебная статистика РФ» (уже подключено)
Агрегированная статистика Суддепартамента в удобном виде: наказания по каждой
статье УК, меры пресечения, основания прекращения дел, гражданские/административные/
арбитражные категории. Выгружено в `knowledge/sudstat/` (2016–2024), обновляется
на сайте раз в полугодие/год — выгрузку можно повторять тем же скриптом.

### ✅ cdep.ru — Судебный департамент при ВС РФ
Первоисточник статистики: официальные формы отчётности (xls) по полугодиям.
Сырее, чем АПИ, но там есть формы, которых нет в агрегаторе (например, детальная
10.3 по мерам наказания). Сайт медленный, но открывается.

## Платные системы (если понадобится глубина)
- **КонсультантПлюс** / **Гарант** — комментарии, актуальные редакции норм,
  аналитика; бесплатные интернет-версии ограничены по времени доступа.
- **Caselook, Casebook** — коммерческий поиск практики с API и мониторингом.

## Рекомендация по автоматизации
1. Адвокат даёт список статей, по которым ведёт дела (например: 171.1, 159, 228…).
2. Ежемесячный cron на сервере: sudact-поиск свежих приговоров/апелляций по
   каждой статье → markdown-дайджест в `knowledge/fresh_practice/<статья>/<YYYY-MM>.md`.
3. Раз в полугодие — повторная выгрузка статистики АПИ (скрипт уже есть).
4. Обзоры ВС РФ — добавлять вручную в `knowledge/` по мере выхода (PDF → текст).

Агенту: когда спрашивают про свежую практику по статье — сначала посмотри
`knowledge/fresh_practice/` и `knowledge/sudstat/`, потом честно скажи, какой
датой ограничены твои материалы.
