# Судебная статистика РФ по всем видам судопроизводства (2016–2024)

Источник: stat.апи-пресс.рф (данные Судебного департамента при ВС РФ), вся Россия,
годы 2016–2024. Выгружено 2026-07-19 скриптом `tools/sudstat_fetch_all.py`
(перезапуск обновит данные, сайт пополняется раз в полугодие/год).

Формат всех файлов: `Год, Метка_норм, Показатель(оригинал), <числовые колонки>`.
Числа — количество дел/лиц. Ищи Grep-ом по колонке Метка_норм.
Статистика наказаний по statьям УК — отдельно: `../sudstat_nakazaniya_po_statyam_2016-2024.csv`.

## Уголовное судопроизводство (ug_*)
- `ug_obshchie_pokazateli_po_kategoriyam.csv` — поступило/рассмотрено/осуждено/оправдано по категориям дел.
- `ug_tyazhest_prestupleniya.csv` — структура осуждённых по тяжести преступления.
- `ug_osnovaniya_prekrashcheniya_dela.csv` — по каким основаниям суды ПРЕКРАЩАЮТ дела (примирение, деятельное раскаяние, судебный штраф…) — ключ для позиции «прекращение».
- `ug_osnovnye_i_dop_nakazaniya.csv` — основные и дополнительные наказания.
- `ug_osnovaniya_osvobozhdeniya_ot_nakazaniya.csv` — освобождение от наказания (амнистия, болезнь, отсрочка…).
- `ug_kontingent_osuzhdennyh.csv` — кто осуждённые (пол, возраст, занятость, судимость).
- `ug_osobennosti_soversheniya.csv` — группа/опьянение/рецидив и т.п.
- `ug_mery_presecheniya.csv` — сколько заключений под стражу/домашних арестов/залогов просят и удовлетворяют — аргументы против стражи.
- `ug_hodataystva_pri_sledstvii.csv` — обыски, прослушка, продления и как часто суды их удовлетворяют.

## Гражданское (gr_*)
- `gr_obshchie_pokazateli_po_kategoriyam.csv`, `gr_otdelnye_kategorii_del.csv` (1878 строк: иски по видам с суммами удовлетворения), `gr_istcy_i_otvetchiki.csv`.

## Административное судопроизводство КАС (adm1_*)
- Общие показатели, отдельные категории (1613 строк), категории истцов/ответчиков.

## КоАП (koap_*)
- `koap_otdelnye_pravonarusheniya.csv` — 3298 строк по статьям КоАП: сколько наказано, штрафы, аресты, лишения прав.
- `koap_mery_obespecheniya.csv`, `koap_kategorii_pravonarushiteley.csv`.

## Арбитраж (arb_*)
- `arb_obshchie_pokazateli.csv` (1436 строк), `arb_otdelnye_kategorii_del.csv`.

Когда обсуждаешь шансы/риски по конкретной категории дел — сначала посмотри
сюда: реальные частоты решений сильнее любых общих рассуждений.
